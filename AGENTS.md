Agentes Especializados
Para tarefas relacionadas ao frontend (componentes, UI, estilização, services), use o agente especializado:
`.codex/agents/frontend.md`

Para tarefas relacionadas ao backend em Go (usecases, ports, adapters, domínio, persistência, rotas HTTP e testes), use o agente especializado:
`.codex/agents/go-agent.md`

Estrutura do Projeto
app/ — Frontend (SvelteKit + TypeScript + Tailwind + shadcn-svelte)
app/components/app/ — Componentes customizados
app/components/ui/ — Componentes shadcn-svelte (não editar)
app/services/ — Comunicação com o backend

backend/ - backend em go
